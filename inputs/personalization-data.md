# Personalization Data Checklist

Data points to research for each prospect. Use this checklist to gather information for AI-powered personalization.

---

## Instructions

1. Use this checklist when researching prospects
2. Prioritize high-impact data points (marked with ⭐)
3. Feed collected data to AI prompts for email generation
4. Store in Smartlead custom fields for mail merge

---

## Research Checklist

### Prospect Information

#### Basic (Required)

- [ ] ⭐ **First name** — For greeting
- [ ] ⭐ **Last name** — For formal reference
- [ ] ⭐ **Email** — Verified, deliverable
- [ ] ⭐ **Company name** — Official name (not domain)
- [ ] ⭐ **Job title** — Current role

#### Professional Background

- [ ] **LinkedIn URL** — Primary research source
- [ ] **Years in current role** — Tenure signal
- [ ] **Previous companies** — Career path context
- [ ] **Education** — Potential connection point
- [ ] **Certifications/Skills** — Expertise areas

---

### Company Information

#### Basic Company Data

- [ ] ⭐ **Industry/Vertical** — For industry-specific messaging
- [ ] ⭐ **Company size** — Employee count or range
- [ ] **Headquarters location** — Geography relevance
- [ ] **Founded year** — Maturity stage
- [ ] **Website** — Research source

#### Business Context

- [ ] ⭐ **Funding status** — Stage, recent rounds
- [ ] **Revenue** — If public/available
- [ ] **Growth stage** — Startup, growth, enterprise
- [ ] **Business model** — B2B, B2C, marketplace, etc.

---

### Trigger Events (Timely Signals)

#### Company Triggers

- [ ] ⭐ **Recent funding** — Series A, B, C, etc.
- [ ] ⭐ **Hiring activity** — Relevant roles being hired
- [ ] **New leadership** — C-suite changes
- [ ] **Product launch** — New offering announced
- [ ] **Expansion** — New markets, offices
- [ ] **Acquisition/Merger** — M&A activity
- [ ] **Press coverage** — Recent news mentions
- [ ] **Award/Recognition** — Industry awards

#### Prospect Triggers

- [ ] ⭐ **Recent LinkedIn post** — Topics they care about
- [ ] **Job change** — New role (within 90 days)
- [ ] **Promotion** — Recently promoted
- [ ] **Speaking engagement** — Conference, podcast
- [ ] **Published content** — Blog, article, report

---

### Pain Point Indicators

#### Role-Based Signals

- [ ] ⭐ **Job postings** — Roles they're hiring for
- [ ] **Team size** — Growing or stable
- [ ] **Responsibilities** — From LinkedIn or job description
- [ ] **Tools mentioned** — In job posts or profile

#### Company-Based Signals

- [ ] ⭐ **Tech stack** — Current tools (BuiltWith, job posts)
- [ ] **Competitor usage** — Using competitor product
- [ ] **Reviews/Complaints** — G2, Glassdoor signals
- [ ] **Customer complaints** — If B2C, public feedback

---

### Personalization Hooks

#### Connection Points

- [ ] **Mutual connections** — Shared LinkedIn connections
- [ ] **Shared groups** — LinkedIn groups, communities
- [ ] **Shared alma mater** — School connections
- [ ] **Shared previous employer** — Company alumni
- [ ] **Conference attendance** — Same events

#### Content-Based Hooks

- [ ] ⭐ **Recent LinkedIn post** — Topic, opinion shared
- [ ] **Article/Blog written** — Thought leadership
- [ ] **Podcast appearance** — Interview topics
- [ ] **Comment on post** — Engagement with others
- [ ] **Shared content** — What they reshare

## Research Workflow

### Quick Research (5 min/prospect)

1. **LinkedIn Profile** (2 min)
   - Current role and tenure
   - Recent posts (last 30 days)
   - Background summary

2. **Company Check** (2 min)
   - Size and industry
   - Recent news (Google "[Company] news")
   - Hiring activity

3. **Document** (1 min)
   - Fill in custom fields
   - Note icebreaker hook

### Deep Research (15 min/prospect)

1. **LinkedIn Deep Dive** (5 min)
   - Full activity history
   - Shared connections
   - Groups and interests

2. **Company Research** (5 min)
   - Crunchbase for funding
   - Careers page for hiring
   - Blog for priorities

3. **External Search** (3 min)
   - Google for press mentions
   - Podcast appearances
   - Published articles

4. **Document & Generate** (2 min)
   - Fill all relevant fields
   - Generate AI icebreaker
   - Note specific hooks

---

## Quality Checklist

Before adding prospect to campaign:

- [ ] Email verified as deliverable
- [ ] Company name matches official name
- [ ] Job title is current (not outdated)
- [ ] At least one personalization hook identified
- [ ] Icebreaker is specific and verifiable
- [ ] Data is recent (within last 6 months)
